About Me

Halo! nama saya Yulius Panduakatidahu seorang mahasiswa yang sedang meniti jalan untuk menjadi ahli dalam dunia percodingan. Saat ini saya tengah menempuh studi di [Universitas Stekom Majapahit 605], dengan fokus utama pada pengembangan web dan teknologi. Saya percaya bahwa teknologi memiliki kekuatan untuk mengubah dunia dan membawa dampak positif dalam kehidupan sehari-hari.

Sejak kecil, saya selalu tertarik dengan bagaimana teknologi bekerja dan berfungsi. Rasa penasaran ini mendorong saya untuk mempelajari lebih dalam tentang pemrograman dan pengembangan web. Selama perjalanan akademik saya, saya berusaha untuk membuat sesuatu, baik itu untuk keperluan tugas kuliah maupun untuk kepuasan pribadi.
